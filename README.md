# v2g App

Client-side transcode video files to gif format.

v2g app uses [FFMPEG.WASM](https://github.com/ffmpegwasm/ffmpeg.wasm#readme), a pure WebAssembly / JavaScript port of [FFmpeg](https://ffmpeg.org).
