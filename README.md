# V2G App

Client-side transcode video files to gif format.

V2G app uses [FFMPEG.WASM](https://github.com/ffmpegwasm/ffmpeg.wasm#readme), a pure WebAssembly / JavaScript port from [FFmpeg](https://ffmpeg.org).
